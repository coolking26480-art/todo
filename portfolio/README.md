# Sujal Jain Portfolio

A production-ready, immersive single-page portfolio built with Next.js 15, Tailwind CSS, and Framer Motion.

## Features

- **5 Views**: Home, Code Projects, Design Portfolio, Substack Publications, About Me
- **Interactive Neuron Canvas**: HTML5 Canvas particle system with bioluminescent synaptic transmission animation
- **Glassmorphism Navigation**: Frosted glass effect with backdrop blur
- **Swiss Grid Design**: Clean, structured typography and layouts
- **Responsive**: Fully responsive across all device sizes
- **Optimized Images**: Next.js Image component for optimal loading

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Import to Vercel
3. Deploy automatically

### Local Development

```bash
npm install
npm run dev
```

### Static Export

```bash
npm run build
```

Output will be in `out/` directory.
